# 定制指标卡3使用说明

## 数据接口字段

```json
[
    {
        name: "text",// 标题文字
        value: "text",
    },
    {
        name: "value",// 得分值
        value: "value",
    },
    {
        name: "title",// 下面标题文字
        value: "title",
    },
    {
        name: "level",// 等级
        value: "level",
    },
],
```

## 数据源

```json
[
    {
        "text": "总体得分",
        "value": 88,
        "title": "总体评价",
        "level": "良"
    }
]
```



## 样式配置

- 表盘
  - 外圆速率：外圆旋转速率
  - 内圆速率：内圆旋转速率
- 指标
  - 数值：
    - 文本样式：可以改变数值的文本样式
    - 偏移：对数值的水平和垂直位置偏移
  - 数值描述：
    - 文本样式：可以改变数值描述的文本样式
    - 偏移：对数值描述的水平和垂直位置偏移
- 标题
  - 文本样式：对标题文本的样式进行改变
  - 等级样式：对等级文本的样式进行改变
  - 偏移：标题的总体偏移
