# 定制指标卡4使用说明

## 数据接口字段

```json
[
    {
        name: "value", //数值
        value: "value",
    },
    {
        name: "unit",//单位
        value: "unit",
    },
    {
        name: "title",//标题
        value: "title",
    },
]
```

## 数据源

```json
[
    {
        "value": 123456,
        "unit": "次",
        "title": "受攻击数量"
    }
]
```



## 样式配置

- 数值
  - 文本样式：可以改变数值描述的文本样式
  - 偏移：对数值描述的水平和垂直位置偏移
- 单位
  - 文本样式：可以改变单位描述的文本样式
  - 偏移：对单位描述的水平和垂直位置偏移
- 标题
  - 文本样式：可以改变标题描述的文本样式
  - 偏移：对标题描述的水平和垂直位置偏移
