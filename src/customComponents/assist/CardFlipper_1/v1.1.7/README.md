# 定制翻牌器1组件使用说明

## 数据接口字段

```json
[
    {
        name: "value",// 数值
        value: "value",
    }
],
```

## 数据源

```json
[
    {
        "value": 123456
    }
]
```



## 样式配置

- 数值
  - 文本样式：可以改变数值描述的文本样式
- 翻牌器
  - 数量：翻牌器的数量
  - 大小：翻牌器的大小
  - 间距：翻牌器之间的间距
- 分割
  - 分割符间距：可以设置是否开启，开启时可以设置分割符左右间距大小。
